## 🚀 Key Projects & Impact
- 🍇 **Fruit Freshness Detection System:** Benchmarked MobileNetV2 (**~100% Test Accuracy**) vs YOLOv8 for edge-device efficiency using computer vision and adaptive preprocessing (CLAHE, HSV/LAB).
- 🚚 **GTSP Route Optimization Engine:** Developed a multi-objective ACO (MOACO) model achieving a **18.95% distance reduction** (20.54 km saved) over traditional greedy routing.
- 📊 **ITB Single Source of Truth Data Infrastructure:** Consolidated fragmented data across 5 core university sectors into a Google BigQuery Data Warehouse, powering live Metabase & executive Power BI dashboards.
- 🍹 **RAMU Personalized Health Drink App:** Designed an ISO-compliant personalized health drink ecosystem based on House of Quality (HoQ), achieving a **94.64 SUS Score (Grade A+)**.
